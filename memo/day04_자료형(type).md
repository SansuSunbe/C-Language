
- 자료형이란?
	-> 저장공간의 종류(type)

- 자료형(종류 - type)
	- 정수형
		- int (기본형)
			-> 4 byte
		- long 
			-> 4(32bit), 8(64bit) byte
	- 실수형
		- float
			-> 4 byte
		- double (기본형)
			-> 8 byte
	- 문자형
		- char
			-> 1 byte 
			-> 'a', 'A', '0' (한글은 2byte이기 때문에 사용불가)
	- 문자열
